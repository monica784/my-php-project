## Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/your-repo.git
cd your-repo
```

### 2. Start XAMPP
- Open XAMPP Control Panel
- Start **Apache** and **MySQL**

### 3. Place Files in htdocs
- Copy this project folder into: `C:/xampp/htdocs/` (Windows) or `/Applications/XAMPP/htdocs/` (Mac)

### 4. Access in Browser
```
http://localhost/my-php-project/
```
